# Amazon Product API

Typescript interface to Amazon product advertising api.

## Usage

> npm install @mcrowe/amazon-product-api --save

## Development

Install npm modules:

> npm install

Run tests:

> npm test

## Release

Release a new version:

> bin/release.sh

This will publish a new version to npm, as well as push a new tag up to github.
